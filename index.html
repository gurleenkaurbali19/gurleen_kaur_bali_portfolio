Personal Chatbot

  • Developed a personalized chatbot using FAISS (Facebook AI Similarity Search) and the SentenceTransformer library.
  • Created a JSON file containing personal Q&A data as prompt-response pairs.
  • Used SentenceTransformer (all-MiniLM-L6-v2) to convert both user queries and stored prompts into vector embeddings.
  • Built a FAISS index to store these vector embeddings for efficient similarity search.
  • When a user asks a question, the chatbot converts the query into an embedding, performs a nearest neighbor search in the FAISS index using Euclidean distance, and retrieves the most relevant response.
  • Used Streamlit to build a clean and interactive frontend interface where users can chat with the bot.
